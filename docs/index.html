Glassine Paper

CSS for Material Design 3
Sample Link
GitHub

Usage

@import url('https://fonts.googleapis.com/css?family=Roboto:300,300i,400,400i,500,500i,700,700i,900,900i');
@import './theme.css';
@import './glassine-paper.css';

Install via npm

npm install https://github.com/MichinobuMaeda/glassine-paper-css

Usage in JavaScript/Node.js projects

import 'glassine-paper-css/glassine-paper.css';

Or with require

require('glassine-paper-css/glassine-paper.css');

In HTML (if using a bundler like webpack/vite)

<link rel="stylesheet" href="node_modules/glassine-paper-css/glassine-paper.css">

For build tools, you can also import the source SCSS

import 'glassine-paper-css/src/index.scss';
